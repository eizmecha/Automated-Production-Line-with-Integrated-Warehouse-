# Automated-Production-Line-with-Integrated-Warehouse-

This project presents the design and implementation of a fully automated manufacturing 
process using Siemens S7-PLCSIM S7_300, programmed through TIA Portal V17, and simulated 
in Factory IO. The project encompasses six key stages: sorting, machining, assembly, packaging, 
storage, and distribution, each integral to the overall workflow of a production line. In the initial 
sorting process, products are identified by color, with green items removed and blue items 
progressing through the line. The blue items undergo a machining process where they are reshaped 
by a CNC machine and then transferred to an assembly station, where a SCARA robot completes 
the final product assembly. Subsequently, the products are packaged and moved to storage before 
being prepared for distribution. Each process is managed and synchronized by a PLC system, 
utilizing sensors, actuators, and a Human-Machine Interface (HMI) for real-time monitoring and 
control. This simulation demonstrates the potential of PLC-based automation in achieving efficient 
and precise production workflows, highlighting its relevance in modern industrial environments.
